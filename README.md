# Starter Project
starter project webpack template

```git clone git@github.com:dominikmietla/Starter-project.git```

Go into directory

```cd Starter-project```

### Assets

Install dependencies

```
npm install
```

Build assets:
```
npm run build
```

Watch assets for changes:
```
npm start
``` 

From now you can access project from url

http://localhost:3000
